== Description ==

Bee Addons Blocks est une extension conçu pour l'éditeur Gutenberg. Cette extension propose une collection de blocs personnalisés soigneusement conçus, offrant une polyvalence accrue dans la création de contenu.

**Fonctionnalités principales :**
- **Blocs personnalisés :** Intégrez des éléments spécifiques adaptés à vos besoins, des mises en page sophistiquées aux fonctionnalités interactives.
- **Compatibilité avec Gutenberg :** Exploitez pleinement le potentiel de l'éditeur de blocs natif de WordPress pour une expérience intuitive de création de contenu.
- **Optimisé pour la performance :** Bee Addons Blocks est développé en suivant les meilleures pratiques de performance, garantissant une expérience utilisateur rapide et fluide.

**Compatibilité :**
Bee Addons Blocks s'intègre harmonieusement avec les normes PSR-12, suivant une approche orientée objet côté serveur. Un typage strict des variables assure la stabilité et la robustesse du code.

Explorez de nouvelles possibilités avec Bee Addons Blocks et améliorez votre expérience de création de contenu sur WordPress !

== Installation ==

Cette section décrit comment installer le plugin et le mettre en marche.

1. Téléchargez les fichiers du plugin dans le répertoire `/wp-content/plugins/bee-addons-blocks`, ou installez le plugin directement depuis l'écran des plugins de WordPress.
2. Activez le plugin depuis l'écran 'Plugins' dans WordPress.
3. Et voilà, c'est tout !
